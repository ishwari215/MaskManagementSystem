Our Vision:

Our primary goal is creating a technical background for the process of mask collection which will solve the problem of increasing biomedical waste in the environment due to the pandemic situation. To create such a software having user friendly interface and a database system keeping track of collected masks. Make people aware about the process of converting masks into bricks which will help in keeping surroundings clean and hygienic.
